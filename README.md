# 🎙️ AI Mock Interview Platform

Welcome to the **AI Mock Interview Platform**, a cutting-edge web application designed to help job seekers master their interview skills through high-fidelity, AI-powered simulations.

Developed by **[Muskan Sindhu](https://github.com/MuskanSindhu25)**.

---

## 🌟 Overview

This platform provides a realistic interview experience by using advanced AI to simulate a professional recruitment process. Candidates can practice for specific job roles, receive real-time feedback, and get a detailed performance analysis at the end of each session.

## 🚀 Key Features

- **Dynamic AI Interviewer**: Engage in real-time conversations with an AI that asks role-specific questions.
- **Role Customization**: Tailor the interview Experience by selecting specific job roles and seniority levels.
- **Voice Interaction**: Integrated Speech-to-Text and Text-to-Speech (via ElevenLabs) for a natural conversational flow.
- **Detailed Analytics**: Receive an AI-generated report highlighting your strengths, areas for improvement, and a total score.
- **Premium UI/UX**: Built with a sleek, modern interface featuring glassmorphism, fluid animations (Framer Motion), and responsive design.

## 🛠️ Tech Stack

- **Frontend**: React.js 19, Vite
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Styling**: Modern CSS / Glassmorphism
- **AI Services**: Google GenAI / ElevenLabs (for voice)

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/MuskanSindhu25/major-project-
   cd major-project-
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   Create a `.env` file in the root directory and add your API keys:
   ```env
   VITE_ELEVENLABS_API_KEY=your_elevenlabs_key
   VITE_GEMINI_API_KEY=your_gemini_key
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```
